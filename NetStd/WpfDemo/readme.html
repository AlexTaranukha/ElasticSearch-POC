dtSearch .NET Standard WPF Sample

Overview

This sample demonstrates how to use the dtSearch Engine's .NET Standard API in a .NET Framework 4.x application.

To run this sample code:

  1. Open the solution in Visual Studio 2017 version 15.5 or later.
  2. Build and run the application
